Development And Usage


I developed this project as a team alongside Cole Branston using git and git based services within a repository. This collaboration project utilizes Android Studio's application development environment, and, therefore, is accessible on any Android device. To use the code provided below you must open it either through Android Studio's desktop application and/or run it on your Android device.


Programming


This application was programmed using Java. The app itself uses the New York Times and AlphaVantage APIs to provide information about particular stocks for the user and general news concerning the biggest stocks in the market.


Features


The Android application consists of a general board to view news, a ticker search page that prompts users for a stock-specific ticker symbol, an account information page that provides general information about the current user, and a signup form with an email verification service.
